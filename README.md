inteiros. Ao final, imprima esses números de forma organizada conforme
exemplo. Em seguida, informar qual foi o primeiro valor informado.

Criar um serviço de impressão:

![](Integer.png)
```
How many values? 3

8
23
[10, 8, 23]
First: 10
```
